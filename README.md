# Car Dealership Website

This is a simple car dealership website built using HTML, CSS, and JavaScript.  
The website displays featured cars, allows users to browse vehicles, and contact the dealership for more information.

## Features

- Homepage with hero section
- Featured car listings
- Car details section
- Contact form
- Responsive layout
- Simple navigation menu

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

Car-dealership-website
│
├── index.html
├── css
│   └── style.css
├── js
│   └── script.js
└── images

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
